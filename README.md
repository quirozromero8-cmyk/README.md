# Análisis de Vehículos Usados en EE.UU.

El fin de esta aplicación web es explorar un conjunto de datos de anuncios de venta de vehículos usados en Estados Unidos. Fue desarrollada con Streamlit y ofrece visualizaciones interactivas sencillas para facilitar el análisis de variables como el precio y el odómetro (distancia recorrida)

## Funcionalidades

- Vista previa de los datos
- Histograma del odómetro
- Gráfico de dispersión entre odómetro y precio
- Interacción mediante botones

## Librerias utilizadas

- Python
- Streamlit
- Pandas
- Plotly

## Cómo ejecutarla

```bash
streamlit run app.py
